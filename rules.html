German verbs with prepositions

As in English, some German verbs are followed by prepositions to give a certain meaning. These are called "verbs with prepositions". For example, "to wait for" would be "warten auf" in German.

How can a person learning German learn which preposition goes with which verb?

Well, as in many challenges, there are many ways. Some people would be able to remember it easily just by reading or listening to German (the lucky ones). Some verbs with prepositions are similar in English and German, but unfortunately not all of them are.

Learning by matching verbs using the same preposition.

Some verbs use the same preposition. For example, the verbs "warten" and "achten" use the preposition "auf".

So why not learn verbs by matching verbs that use the same preposition?

That's what this site is all about.

The Memo Game

It is inspired by memory card games where the player has to turn over cards to find pairs of identical pictures. Instead of finding pairs of pictures, the player of the German Memo Game has to find pairs of verbs with the same preposition.

Quiz game

The quiz has the particularity that in case of a wrong answer, the player will receive a feedback with the right answer (the right preposition) and also a list of verbs that have the same preposition.

Limits of the games

Accusative/Dative

In German, verbs with prepositions can be followed by the accusative or the dative. In this game, the accusative or dative is not taken into account.

Verbs with more than one preposition

Some German verbs can be followed by different prepositions giving different meanings. For example: sich freuen can be followed by the prepositions auf or über. sich freuen auf = to look forward to, while sich freuen über = to be pleased. These verbs were not used in the game.
